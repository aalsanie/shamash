Reports class names that violate explicit and meaningful naming conventions.

This inspection helps detect generic, abbreviated, or misleading class names that obscure intent and weaken architectural clarity.

Banned suffixes such as Manager, Helper, and Impl are flagged, as they often hide responsibility instead of expressing it. Shortened or abbreviated class names are also reported to encourage self-descriptive design.

See https://plugins.jetbrains.com/docs/intellij/inspections.html#descriptions for more information.

Example of violations: 


// Generic and banned suffix
public class UserManager {
}

// Abbreviated and unclear
public class AuthSvc {
}

Clear naming establishes architectural intent and reduces the need for additional documentation. This inspection promotes domain-driven and intention-revealing class names.
